- [ ] Ran `helpers/check_env.ps1`
- [ ] Expert count recorded

- [ ] Cluster A objective stated
- [ ] Cluster B objective stated
- [ ] Agent count for each cluster is 1-5
- [ ] Industrial safety mode is `proposal_only`

## Phase 2

- [ ] User selected execution mode
- [ ] User approved creating both clusters
- [ ] Cleanup policy selected
- [ ] No agent creation before approval

## Phase 3

- [ ] Cluster A created
- [ ] Cluster B created
- [ ] Each cluster received only its scoped task
- [ ] Cluster outputs summarized
- [ ] Raw long outputs stored or referenced by file path
- [ ] Industrial cluster did not produce direct control commands

## Phase 4

- [ ] Cluster A reviewed industrial templates for schema and helper compatibility
- [ ] Cluster B reviewed framework outputs for safety compliance
- [ ] Final summary written
- [ ] Cleanup action executed or explicitly deferred
- [ ] Lessons learned recorded
