# Offer Catalog

This catalog is for early revenue experiments. Prices are starting hypotheses,
not fixed business policy.

## Offer 1: AI Workflow Audit

Target buyer:

- Individual developers
- Small operations teams
- Content or consulting teams with repeated manual work

Deliverables:

- Current workflow map
- Bottleneck analysis
- Tool and data boundary notes
- 1-5 agent deployment suggestion
- First-run prompt and checklist

Suggested starting price:

- RMB 499 to 1999 for a lightweight audit
- RMB 2000 to 5000 when implementation guidance is included

Why this first:

- Low liability
- Fast delivery
- Directly uses the Master Orchestrator framework

## Offer 2: Agent Pack Setup

Target buyer:

- Users already experimenting with Codex, OpenClaw, Claude Code, or similar tools
- Teams that want repeatable multi-agent workflows but do not want to design the
  orchestration layer themselves

Deliverables:

- 1-5 agent pack template
- Master start prompt
- Phase checklist
- Cleanup and archive policy
- Test scenario and scorecard

Suggested starting price:

- RMB 3000 to 8000 for one focused workflow
- RMB 8000 to 20000 for a workflow family with documentation

Scope boundary:

- No guaranteed model performance
- No credential handling
- No uncontrolled tool execution

## Offer 3: Industrial Agent Safety Review

Target buyer:

- Industrial software teams
- Automation integrators
- Manufacturing digitalization consultants

Deliverables:

- Proposal-only architecture review
- Safety redline checklist
- Human approval matrix
- Tool whitelist design
- Audit trail field list
- Edge fallback and degradation notes

Suggested starting price:

- RMB 5000 to 20000 for document-level review
- Higher pricing only after domain evidence and expert review are available

Scope boundary:

- No live equipment control
- No PLC/DCS parameter advice
- No replacement for functional safety assessment
- No claim of IEC/SIL compliance unless reviewed by qualified experts

## Offer 4: Research And Report Pack

Target buyer:

- Consultants
- Product managers
- Agencies
- Founders preparing a niche strategy

Deliverables:

- Structured research report
- Executive summary
- Risk and assumption list
- Source checklist
- Optional slide outline

Suggested starting price:

- RMB 999 to 3000 for a narrow report
- RMB 3000 to 10000 when multiple iterations and packaging are included

## Recommended First Offer

Start with Offer 1 or Offer 2.

They are closest to the current repo, easiest to demonstrate, and do not require
industrial customer trust on day one.
